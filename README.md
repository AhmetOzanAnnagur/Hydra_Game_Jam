# Hydra - Bilkent GameJam 26

A game project developed for Bilkent GameJam 26 using Unity.

## 📋 Project Overview

Hydra is an interactive game built with Unity, featuring rich animations, audio design, and a complete gameplay experience.

## 🎮 Features

- **Animations**: Smooth character and object animations
- **Audio Design**: Immersive sound effects and music system
- **Tilemap-based Gameplay**: Level design using Unity Tilemaps
- **UI System**: Complete user interface with TextMesh Pro
- **Input System**: Modern input handling with Unity's Input System
- **Graphics**: Universal Render Pipeline (URP) for optimized rendering
- **Resource Management**: Organized asset structure

## 📁 Project Structure

```
Assets/
├── Animations/      - Character and object animations
├── Audio/           - Sound effects and music
├── Hydra/           - Game-specific assets
├── Prefabs/         - Reusable game objects
├── Resources/       - Dynamic resource loading
├── Scenes/          - Game levels and scenes
├── Scripts/         - C# game logic
├── Settings/        - Project configuration
├── Sprites/         - 2D graphics and sprites
├── Tilemap/         - Tilemap data and layers
└── TextMesh Pro/    - Text rendering assets
```

## 🚀 Getting Started

### Requirements
- Unity (version specified in ProjectSettings/ProjectVersion.txt)
- .NET Framework

### Installation

1. Clone this repository:
```bash
git clone https://github.com/AhmetOzanAnnagur/Hydra_Game_Jam.git
```

2. Open the project in Unity:
   - Launch Unity Hub
   - Click "Add" and select the project folder
   - Open the project

3. Wait for Unity to import all assets

### Running the Game

- Open a scene from `Assets/Scenes/`
- Press **Play** in the Unity Editor
- Use the configured input system to control the game

## 🔧 Building

To create a standalone build:
1. Go to File → Build Settings
2. Select your target platform (Windows, macOS, Linux, etc.)
3. Click "Build" and choose an output folder

A pre-built version for Windows is available in the `game build/` folder.

## 📝 Notes

- The project uses Unity's Input System for improved input handling
- Universal Render Pipeline (URP) is configured for better performance
- TextMesh Pro is configured for UI text rendering
